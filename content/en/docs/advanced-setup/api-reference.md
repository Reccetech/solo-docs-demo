## API Reference

Use the following URL pattern to open the Solo API reference for a specific release:

`https://solo.hiero.org/v<SOLO_VERSION>/classes/`

For example, for version `0.57.0`:

`https://solo.hiero.org/v0.57.0/classes/`

To view another release, replace `0.57.0` with the version you want.
